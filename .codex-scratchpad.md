# Polyfacts Codex Scratchpad

## Project
Political fact-checking platform: audio transcription → claim detection → evidence retrieval → verdict generation.

## Stack
- Backend: Python 3.12, FastAPI, PostgreSQL + pgvector, Redis + arq, SQLAlchemy 2.0, Alembic
- Frontend: Next.js 14, React 18, TypeScript, Tailwind CSS
